# DJ Playlist Exporter

iTunesプレイリストをUSB用フォルダへ簡単に書き出せるMac向けツールです。

プレイリストの順番を維持したまま楽曲ファイルをコピーできるため、DJ前のUSB作成を効率化できます。

---

## 主な機能

- プレイリスト順で書き出し
- BPM順で書き出し
- プレイリスト検索
- プレイリスト一覧表示
- 保存フォルダ名の自動入力
- 保存先の空き容量チェック
- 完了後にフォルダを開く
- 完全ローカル動作

---

## 書き出し例

### プレイリスト順

01 Intro.mp3  
02 Track A.mp3  
03 Track B.mp3

### BPM順

90 Intro.mp3  
95 Track A.mp3  
128 Track B.mp3

BPM未登録の楽曲は曲名のみで出力されます。

## 対応ファイル

iTunesライブラリ内のローカル楽曲

- MP3
- AAC (.m4a)
- WAV
- AIFF

※ DRM保護された楽曲、ストリーミング楽曲は対象外です。

## 起動方法

### Mojave（10.14）の場合

1. DJ Playlist Exporter.app を右クリック

2. 「開く」を選択

3. 警告が表示されたら「開く」をクリック

次回以降はダブルクリックで起動できます。


### Catalina（10.15）以降の場合

ターミナルを使った手順が必要です。

1. ターミナルを開く
   Finder → アプリケーション → ユーティリティ → ターミナル

2. 以下の文字を入力（最後にスペースを入れてください）
   xattr -rd com.apple.quarantine

3. DJ Playlist Exporter.app を
   ターミナルのウインドウにドラッグ＆ドロップ

4. Enter キーを押す

5. DJ Playlist Exporter.app をダブルクリックで起動

この手順は初回のみ必要です。
